typing_meter
======

.. figure:: _static/demo.gif
   :align: center

   Terminal based typing speed tester

**typing_meter** is a program to test (and hence improve) your typing speed right from
the ease of your terminal.

Main Features
-------------

- 🖥️ Cross-platform
- 🎦 See your replay
- 🐍 Supports Python 2 and 3
- 📝 Choose custom text input
- 🅰️ 6000 text samples
- 🌈 Colored texts
